# Repo rules

## Live product rules
- One live app shell only
- Bottom nav = Home / Calendar / Challenge
- Profile opens from avatar only
- Goal Detail tabs = Plan / Goal Room / People
- No Notes tab
- Visibility = public / private only
- Default visibility = public
- Stream Chat is not part of the live product path
- Malayalam must be supported

## Working rules
- Work only on the live flow
- Do not preserve deprecated flows "for safety"
- Do not use outdated code from another branch
- Keep frontend, backend, types, and rules consistent
- Run build and typecheck after changes
- If something is incomplete, say it clearly
- Reply briefly
